# broken_mirror

This repository contains the raw data and code for the paper **A Broken Mirror: Left-Handers' Serve Placement in Professional Tennis Is Strongly Asymmetric but No Longer Advantageous**.

## Repository contents

Raw data are stored in the split archive _tennis_cache.part01.rar / _tennis_cache.part02.rar (extract by pointing your tool at part01), and the core analysis code is in the Python (*.py) files. Any additional resources will also be released here in the future.

## Requirements

- Python 3.11
- pandas, NumPy, SciPy, statsmodels

```bash
pip install pandas numpy scipy statsmodels
```

## Data sources

Match data are credited to Jeff Sackmann / Tennis Abstract (https://www.tennisabstract.com/) and the Match Charting Project.
